# Redlock

An implementation of the [Redlock] [1] algorithm.

**WARNING** this package is still on development.

## Requirements

* PHP >= 5.4
* Redis

## Adapters

This library is provided with two different adapters:

* [Predis] [2]
* [PHP Redis extension] [3]
